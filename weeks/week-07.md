# Week 7: Interface
*Mar 9*

## In Class
- Interface conventions: cursor, hover, click, drag, scroll, pinch, swipe
- Technical workshop: 2D collage, working with COMP, making your own interface

## Assignment

**Sketch:** Take one existing interface (analog or digital), analyze its layout, and re-interpret it in Touchdesigner. Capture the structure and feeling, not every detail.

- Step 1: a photo/screenshot of an interface you picked
- Step 2: a hand sketch of its layout
- Step 3: recreate it with COMPs, including container, button, slider, etc.
- Bonus: make one or more element interactive
