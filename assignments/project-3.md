# Project 3: Final Project

**Assigned:** Week 11
**Due:** Week 14 (May 4)

## Brief

In this class we have been asking: what can a machine notice and reveal about us and our relationships? It might make the geometry between two people visible, or change when someone leaves, or reward stillness, or encourage a behavior. 

Solo or with a partner, build an system that reveals something about a body part in space, in installation format. Your project should make it clear without you given verbal instructions; a stranger would be able to walk up to it and discover what it does. 

## Requirements

- A defined interaction zone, considering proximics (even if invisible)
- A display* positioned deliberately
- A TouchDesigner system that runs unattended for the duration of a show
- Clear to viewer without spoken instruction

*display formats
- Standard: Each project gets a TV or monitor, positioned vertically or in a configuration you've chosen intentionally. Small speakers or headphones. 
- Projector (requires application): Limited slots, on application basis; will be shown in rotation. Submit a one-paragraph pitch explaining what the setup makes possible that a TV can't.
- Other: Considered also on application basis


## Deliverables

1. **Final presentation** (Week 14)
2. IMA Show installation
3. A score / artist note (due with installation)
4. Documentation (due week after show)
